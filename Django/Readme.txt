To run the site:
*You should have python and browser in your computer
1.Open the terminal
2.Move to the directory DJANGO-->FirstWeb
3.Write in your terminal command:
python manage.py runserver
4.Copy a server link:
For example: http://127.0.0.1:8000/
5.Now go to the browser and paste the link
6.You can move to the second page if you append to the html:
/info/
7.Enjoy!
